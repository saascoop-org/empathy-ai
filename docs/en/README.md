# EmpathyAI

**AI support for neurodivergent people in communication, decision-making and workplace inclusion.**

## Purpose

EmpathyAI is an official SaaSCoop project prepared for long-term open source development.

## Origin

This project continues work originally initiated during the Google Kaggle Gemma 4 Good Hackathon.

The original hackathon repository and Kaggle write-up remain preserved as historical records:

- Historical GitHub repository: https://github.com/HackathonBrTeam/Empathy-Interactional-Expertise
- Kaggle write-up: https://www.kaggle.com/competitions/gemma-4-good-hackathon/writeups/empathyai-cognitive-mediator

## Repository strategy

SaaSCoop preserves the HackathonBrTeam repository as a historical archive of the submitted version.

This repository is intended for the official long-term evolution of the project.

## Status

- Project stage: Incubation
- Maintainer organization: SaaSCoop
- Documentation language model: PT-BR, EN and ES
