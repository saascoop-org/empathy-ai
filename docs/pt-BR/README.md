# EmpathyAI

**IA de apoio a pessoas neurodivergentes na comunicação, tomada de decisão e inclusão no trabalho.**

## Objetivo

O EmpathyAI é um projeto oficial da SaaSCoop preparado para evolução open source de longo prazo.

## Origem

Este projeto dá continuidade ao trabalho iniciado durante o Google Kaggle Gemma 4 Good Hackathon.

O repositório original do hackathon e o write-up do Kaggle permanecem preservados como registros históricos:

- Repositório histórico no GitHub: https://github.com/HackathonBrTeam/Empathy-Interactional-Expertise
- Write-up no Kaggle: https://www.kaggle.com/competitions/gemma-4-good-hackathon/writeups/empathyai-cognitive-mediator

## Estratégia de repositório

A SaaSCoop preserva o repositório HackathonBrTeam como arquivo histórico da versão submetida.

Este repositório é destinado à evolução oficial e permanente do projeto.

## Status

- Estágio do projeto: Incubação
- Organização mantenedora: SaaSCoop
- Modelo de documentação: PT-BR, EN e ES
