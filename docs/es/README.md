# EmpathyAI

**IA de apoyo a personas neurodivergentes en comunicación, toma de decisiones e inclusión laboral.**

## Objetivo

EmpathyAI es un proyecto oficial de SaaSCoop preparado para evolución open source a largo plazo.

## Origen

Este proyecto continúa el trabajo iniciado durante el Google Kaggle Gemma 4 Good Hackathon.

El repositorio original del hackathon y el write-up de Kaggle permanecen preservados como registros históricos:

- Repositorio histórico en GitHub: https://github.com/HackathonBrTeam/Empathy-Interactional-Expertise
- Write-up en Kaggle: https://www.kaggle.com/competitions/gemma-4-good-hackathon/writeups/empathyai-cognitive-mediator

## Estrategia de repositorio

SaaSCoop preserva el repositorio HackathonBrTeam como archivo histórico de la versión enviada.

Este repositorio está destinado a la evolución oficial y permanente del proyecto.

## Estado

- Etapa del proyecto: Incubación
- Organización mantenedora: SaaSCoop
- Modelo de documentación: PT-BR, EN y ES
